# Finance-Dashboard-Modern-UI-Local-Storage-Charts
A simple, clean, responsive finance dashboard built with HTML, CSS, JavaScript, and Chart.js. It includes transaction management with localStorage, dynamic search filtering, role-based UI control, and interactive charts.
A clean, responsive, and lightweight finance dashboard built entirely with HTML, CSS, Vanilla JavaScript, and LocalStorage — no frameworks, no backend.
Track transactions, view summaries, analyze spending patterns, and visualize insights using Chart.js.

Features

Real-Time Transaction List
Add new transactions
Search and filter
Stored in the browser using LocalStorage
Automatically re-renders without page refresh
Interactive Charts
Line chart for balance trend
Pie chart for category breakdown
Built using Chart.js CDN
Clean, Modern UI
Soft card shadows
Minimalistic stat cards
Insight boxes
Fully responsive layout 4. Role-based UI
Admin mode → Add transactions
Viewer mode → Read-only
Persistent Data (Local Storage)
Tech-:
Your added transactions stay saved even after closing the browser.
Structure- HTML5
Styling - CSS3
Logic - Vanilla JavaScript
Storage - LocalStorage
Data Visualization - Chart.js

What Each Folder/ File Does

index.html:-The main HTML file that loads your styles, JS, and components.

css-:All styles are grouped inside one folder for easier maintenance.

style1.css → Main layout + dashboard styling
header.css → Navbar + topbar styles
general.css → Resets, typography, spacing utilities, root variables

js-:All JavaScript files go here.

app1.js → Logic for:
✔ LocalStorage
✔ CRUD transactions
✔ Search filter
✔ Render functions
✔ Role switcher (Admin/Viewer)
